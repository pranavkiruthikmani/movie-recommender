# movie-recommender
A full-stack web application designed to provide personalized movie suggestions based on user input. This project bridges a machine learning backend with a modern, animated frontend to create a seamless user experience.

****Installation & Setup****
1. Clone the repository
2.  **Backend Configuration**: 
    Navigate to the backend folder and ensure Python is installed. Install required data science libraries like `pandas` and `NumPy`.
3.  **Frontend Configuration**: 
    Navigate to the frontend folder and run `npm install` to set up React and Framer Motion dependencies.
4.  **Run Application**: 
    Start the backend server followed by the React development server.
